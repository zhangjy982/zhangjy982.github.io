Octive基本操作

Posted on 2018-07-20

Octive基本操作

一、基本运算

1.基本四则运算

(1)5+6
加法运算，ans = 11；
(2)6-9
减法运算，ans = -3；
(3)65*8
乘法运算，ans = 520；
(4)23/8
除法运算，ans = 2.87500000000000；
(5)4^6
乘方运算，ans = 4096；
(6)sqrt(10)
平方根运算，ans = 3.16227766016838；

2.逻辑运算

(1)1 == 2
判断相等，ans = 0；
(2)1 ~=2
判断不等，ans = 1；
(3)1 && 0
判断逻辑与，ans = 0；
(4)1 || 0
判断逻辑或，ans = 1；
(5)xor(1,0)
判断异或，ans = 1；

二、基本函数

(1)disp(a)
展示变量信息；
(2)disp(sprintf(‘2 decimals: %0.2f’,a))：
显示字符串变量，其中格式和c类似；
(3)PS1(‘>> ‘)
更改左边显示样式；
(4)format long/short
数据格式更改；
(5)hist(w)
将矩阵表示的分布用直方图显示出来，其中w为服从高斯分布的数据组成的矩阵;
(6)hist(w,50)
可以指定生成的直方图的方格的数量；

三、矩阵表示

1.普通表示

(1)A = [1 2;3 4;5 6]
表示一个3x2的矩阵，分号表示一行；

2.行向量表示

(1)V = 1：0.1：2
表示从1开始每次增加0.1，最大到2的行向量；
(2)V = 1：6
表示1 2 3 4 5 6组成的行向量；

3.特殊函数表示

(1)W = ones（2，3）
表示只包含1的一个2行3列矩阵；
(2)W = 2*ones（2，3）
表示只包含2的2行3列矩阵；
(3)W = rand（1，3）
表示包含0-1之间随机小数的1行3列矩阵；
(4)W = randn（2，3）
表示服从高斯分布，标准差或者方差为1的数据组成的2行3列矩阵；

4.单位矩阵表示

(1).E = eye（4）表示生成4x4的单位矩阵；

四、移动数据

1.函数

(1)size(A)
返回矩阵A的各个维度数目的行向量，ans = 3 2；
(2)size(A,1)
返回矩阵A的第一维数目，ans = 3；
(3)size(A,2)
返回矩阵A的第二维数目，ans = 2；
(4)length(V)
返回矩阵的各个维度中的最大值，通常用于返回向量的维度值，ans = 4；
(5)load featuresX.dat
加载数据文件，加载完成后生成与文件名相同名字的变量；
(5)load(‘featuresX.dat’)
加载数据文件的另一种方法；
(6)who
显示所有变量；
(7)whos
显示所有变量及其详细信息；
(8)clear featuresX
删除变量；
(9)v = featuresX(1:10)
将featuresX中第一列数据的1-10个赋值给变量v；
(10)save hello.mat v
使变量v保存为一个名为hello.mat的文件，该文件使用二进制压缩的方式存储；
(11)clear
删除当前工作空间中的所有变量；
(12)load hello.mat
加载之前生成的文件，加载完成后变量v重新自动出现；
(13)save hello.txt v -ascii
将变量v保存为一个名为hello.txt的文本文件；

2.数据获取及移动（A = [1 2 ; 3 4 ; 5 6]）

(1) A(3,2)
返回矩阵A第三行第二列的元素，ans = 6；
(2)A(2,:)
返回第二行所有元素；
(3)A(:,2)
返回第二列所有元素；
(4)A([1 3],:)
返回第一行和第三行所有元素；
(5)A(:,2) = [10;11;12]
将矩阵A的第二列重新赋值；
(6)A = (A,[100;101;102])
为矩阵A增加一列新的向量；
(7)A = (A;[1 3])
为矩阵A增加一行新的向量；
(8)A(:)
将矩阵A中的所有元素形成一个列向量；
(9)C = [A B]
将矩阵A与矩阵B横向合并；
(10)C = [A,B]
将矩阵A与矩阵B横向合并；
(11)C = [A;B]
将矩阵A与矩阵B纵向合并；

五、矩阵数据计算

1.矩阵基本运算(A = [1 2 ;3 4;5 6],v = [1 15 2 0.5])

(1)A B
矩阵乘法，按照正常的矩阵乘法流程运算，运算结果是线性代数矩阵相乘结果；
(2)A . B
矩阵A和矩阵B的各项分别相乘，必须是同型矩阵才可以相乘，点号通常用于表示元素之间的运算；
(3)A .^ 2
矩阵A中的每一个元素都做乘方2的运算；
(4)1 ./ A
矩阵A中的每一个元素都取倒数；
(5)log(A)
矩阵A中的每一个元素取log值；
(6)exp(A)
以e为底数，A中元素为指数的幂运算，结果仍为矩阵；
(7)abs(A)
将矩阵A中的每一个元素都取其绝对值，结果仍为矩阵；
(8)-A
将矩阵中的每一个元素都取其相反数，结果仍为矩阵；
(9)v + ones(length(v),1)
将v中的每一个元素都加1；
(10)v + 1
将v中的每一个元素都加1；
(11)A’
A的转置；
(12)A<3
将矩阵A中的每一个元素和3比较如果满足小于3的条件置一，否则置零，最后形成一个和A同型的矩阵，
矩阵的每个元素是0或者1；

2.矩阵运算中的部分重要函数(A=magic(4))

(1)magic(4)
生成一个幻方矩阵，幻方矩阵的特点的特点是每一行、每一列和对角线上的元素加起来数值相等；
(2)max函数
a.[val,ind] = max(A)
val返回每一列中最大的元素值，并组成一个行向量，ind返回最大元素值的索引值，并组成一个行向量；
b.max(A,[],1)
返回每一列中的最大元素值，并组成一个行向量；
c.max(A,[],2)
返回每一行中的最大元素值，并组成一个列向量；
d.val = max(A)
将val赋值为每一列最大值组成的行向量；
e.max(max(A))
返回矩阵中的最大值；
f.max(A(:))
先将矩阵A中的所有元素转为列向量之后取最大值，也是起到取矩阵最大值的效果；
(3)find(A<3)
返回矩阵A 中元素值小于3的索引，产生顺序为按照列进行，结果为一个列向量；
(4)[r,c] = find(A<=7)
查找矩阵A 中小于等于7的所有元素的索引，r为行索引，c为列索引，查找顺序按照列进行；
(5)sum函数
a.sum(A)
将矩阵A中每一列的元素值求和，返回求和值组成的一个行向量，向量维度和A的列数相等；
b.sum(A,1)
将矩阵A中每一列的元素值求和，返回求和值组成的一个行向量，向量维度和A的列数相等；
c.sum(A,2)
将矩阵A中每一行的元素值求和，返回求和值组成的一个列向量，向量维度和A的行数相等；
d.sum(sum(A . eye(4)))
返回矩阵A正对角线元素之和；
e.sum(sum(A .flipud(eye(4))))
返回矩阵A反对角线元素之和；
(6)flipud(A)
使矩阵A垂直翻转，即最后一行变成第一行，倒数第二行变成第二行，倒数第三行变成第三行，以此类推；
